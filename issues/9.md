## 概要

自治会・地域固有の情報をもとにAIが住民の質問に多言語で答える「地域暮らしサポートAI」機能。
ゴミの捨て方・地域ルール・防災情報など、外国人住民を含むすべての住民の生活を支援する。

---

## 対応する質問カテゴリ

### 🗑️ ゴミ・生活ルール
- 「この袋は燃えるゴミ？」
- 「粗大ゴミはどこに出す？」
- 「今週の収集日はいつ？」
- 「ペットボトルのキャップは外す？」

### 🏘️ 地域のルール
- 「騒音の苦情はどこに言う？」
- 「自治会費はいくら？いつ払う？」
- 「草刈りの当番はいつ？」

### 🚨 防災・緊急
- 「近くの避難所はどこ？」
- 「避難レベル3って何をすればいい？」
- 「非常用持ち出し袋に何を入れる？」

### 🌍 外国人住民向け
- 分別ルールは国によって全然違う → 母国語で正確に案内
- 日本の自治会文化の説明
- 行政手続きの案内

## 技術方針：RAG（検索拡張生成）

ハルシネーション（でたらめ回答）を防ぐため、自治会が登録したデータのみを参照して回答する。

```
自治会がデータを登録
  ├─ ゴミ収集カレンダー（CSV）
  ├─ 地域ルール文書（PDF / テキスト）
  ├─ よくある質問集
  └─ 施設情報・連絡先
       ↓
AIがこのデータを参照して回答生成
       ↓
OpenAI API / Gemini API
```

自治会ごとに独立した知識ベースを持つ（Issue #11 マルチテナントと連携）。

```json
{
  "tenant_id": "shibuya-chome1",
  "knowledge": [
    { "type": "garbage", "content": "燃えるゴミ: 火・金曜..." },
    { "type": "rule", "content": "自治会費: 年3,000円..." },
    { "type": "facility", "content": "避難所: 渋谷区立体育館..." }
  ]
}
```

## UI構成

### 住民側（チャット画面）
- よくある質問（ボタン選択式） ← 高齢者向け・文字入力不要
- 自由入力（テキスト・音声）
- 言語切替ボタン
- 「担当者に聞く」ボタン（AIが答えられない時に自治会担当者へ転送）

### 管理者側
- 知識ベース編集（ゴミ日程・ルール等）
- よくある質問の追加・編集
- AIが答えられなかった質問ログの確認 → 知識ベース改善に活用

## 差別化ポイント

| 一般的なAIチャット | このアプリのAI |
|------------------|--------------|
| 一般的な知識しか答えられない | **その地域固有の情報**で答える |
| 日本語のみ | **多言語対応** |
| 単独アプリ | 回覧板・安否確認と**一体化** |
| 全国共通 | **自治会ごとにカスタマイズ** |
